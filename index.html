introduction

KyriLib is a modern UI library for Roblox scripts. clean interface with smooth animations, built-in config system, and rich UI elements.

get started github

features

  • modern interface with smooth animations
  • built-in save/load config system
  • custom notifications with fade animations
  • mobile support with touch controls
  • buttons, toggles, sliders, inputs, dropdowns, keybinds
  • customizable themes with 9 color options
  • optional icons for tabs
  • draggable and resizable windows
  • keybind system with hold to interact support

installation

load the library:

local KyriLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Justanewplayer19/KyriLib/refs/heads/main/source.lua"))()

basic usage

-- create window
local window = KyriLib.new("My Script", {
    GameName = "MyGame"
})

-- create tab
local main = window:tab("Main")

-- add button
main:button("Click Me", function()
    print("clicked")
end)

-- add toggle
main:toggle("Enable Feature", false, function(state)
    print("state:", state)
end, "feature_flag")

-- send notification
window:notify("Success", "script loaded", 3)

window

creating a window

KyriLib.new(title, options)

parameters

parameter type description
title string window title
options table optional settings (GameName, Theme)

custom theme

customize colors by passing a Theme table:

local window = KyriLib.new("My Script", {
    GameName = "MyGame",
    Theme = {
        bg = Color3.fromRGB(8, 8, 10),
        container = Color3.fromRGB(14, 14, 18),
        element = Color3.fromRGB(20, 20, 26),
        hover = Color3.fromRGB(28, 28, 36),
        active = Color3.fromRGB(32, 32, 42),
        accent = Color3.fromRGB(138, 116, 249),
        text = Color3.fromRGB(245, 245, 250),
        subtext = Color3.fromRGB(165, 165, 180),
        border = Color3.fromRGB(32, 32, 40)
    }
})
info

only specify colors you want to change. unspecified colors use default theme. all 9 color options shown above.

change accent color

window:accent(Color3.fromRGB(255, 100, 100))

mobile support

KyriLib automatically detects mobile and adjusts UI. mobile users get a toggle button in bottom right, PC users press K to toggle.

tabs

creating a tab

window:tab(name, icon)

parameters

parameter type description
name string tab name
icon string (optional) icon name

example with icons

local weapons = window:tab("Weapons", "sword")
local settings = window:tab("Settings", "settings")
icon names

available icons: sword, move, user, music, settings

elements

button

tab:button(text, callback)

creates a clickable button.

toggle

tab:toggle(text, default, callback, flag)

creates a toggle switch. if flag provided, state is saved to configs.

slider

tab:slider(text, min, max, default, callback, flag)

creates a slider for number values. on mobile, shows +/- buttons instead of draggable slider.

input

tab:input(text, placeholder, callback, flag)

creates a text input box.

dropdown

tab:dropdown(text, options, default, callback)

creates a dropdown menu with selectable options.

warning

dropdowns don't save to configs automatically.

label

tab:label(text)

creates a text label for organizing sections.

example

local main = window:tab("Main")

main:label("player options")

main:button("Reset", function()
    print("reset")
end)

main:toggle("Fly", false, function(state)
    print("fly:", state)
end, "fly")

main:slider("Speed", 16, 200, 16, function(value)
    print("speed:", value)
end, "speed")

main:input("Username", "enter name", function(text)
    print("name:", text)
end, "username")

main:dropdown("Location", {"Spawn", "Base", "Shop"}, "Spawn", function(value)
    print("selected:", value)
end)

keybinds

create keybind

tab:keybind(text, default, hold_to_interact, callback, flag)

parameters

parameter type description
text string keybind label
default string default key (e.g. "Q", "LeftShift")
hold_to_interact boolean if true, callback gets true/false when holding
callback function function called when key pressed
flag string optional flag for config saving

examples

press to toggle

main:keybind("Fly Key", "Q", false, function()
    print("fly toggled")
end, "fly_key")

hold to activate

main:keybind("Speed Boost", "LeftShift", true, function(holding)
    if holding then
        print("boost active")
    else
        print("boost inactive")
    end
end, "boost_key")
info

click the keybind box and press any key to rebind. new key is automatically saved if flag provided.

notifications

send notification

window:notify(title, text, duration)

parameters

parameter type description
title string notification title
text string notification message
duration number duration in seconds (default: 3)

example

window:notify("Success", "feature enabled", 2)
window:notify("Error", "something went wrong", 5)

config system

KyriLib automatically saves and loads settings for elements with flags. a settings tab is automatically created with save/load functionality.

accessing flags

-- get value
local speed = window.flags.speed_flag

-- set value programmatically
window.flags.fly_flag_set(true, true)
info

configs are saved per-game using the GameName option. users can save multiple configs with different names.

complete example

local KyriLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Justanewplayer19/KyriLib/refs/heads/main/source.lua"))()

local window = KyriLib.new("prison life", {
    GameName = "PrisonLife"
})

local main = window:tab("Main", "home")
local combat = window:tab("Combat", "sword")

main:label("movement")

main:toggle("fly", false, function(state)
    if state then
        window:notify("Movement", "fly enabled", 2)
    else
        window:notify("Movement", "fly disabled", 2)
    end
end, "fly_toggle")

main:slider("walkspeed", 16, 200, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end, "walkspeed")

main:dropdown("teleport to", {"spawn", "prison", "criminal base"}, "spawn", function(value)
    window:notify("Teleport", "teleported to " .. value, 2)
end)

combat:label("weapons")

combat:button("give all weapons", function()
    window:notify("Weapons", "gave all weapons", 2)
end)

combat:dropdown("gun mode", {"normal", "rapid fire", "one shot"}, "normal", function(value)
    window:notify("Gun Mode", "set to " .. value, 2)
end)

window:accent(Color3.fromRGB(138, 116, 249))
