Introduction

KyriLib is a modern, easy-to-use UI library for Roblox scripts. It provides a clean interface with smooth animations, a built-in config system, and rich UI elements.

Get Started GitHub

Features

  • Modern, clean interface with smooth animations
  • Built-in save/load config system
  • Custom notifications with fade animations
  • Rich elements: buttons, toggles, sliders, inputs, dropdowns
  • Customizable themes
  • Optional Lucide icons for tabs
  • Draggable and resizable windows

Installation

Load the library using the following code:

local KyriLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Justanewplayer19/KyriLib/refs/heads/main/source.lua"))()

Basic Usage

-- Create a window
local window = KyriLib.new("My Script", {
    GameName = "MyGame"
})

-- Create a tab
local main = window:tab("Main")

-- Add a button
main:button("Click Me", function()
    print("Button clicked!")
end)

-- Add a toggle
main:toggle("Enable Feature", false, function(state)
    print("Toggle state:", state)
end, "feature_flag")

-- Send a notification
window:notify("Success", "Script loaded!", 3)

Window

Creating a Window

KyriLib.new(title, options)

Parameters

Parameter Type Description
title string The window title
options table Optional settings (GameName, Theme)

Custom Theme

You can customize the colors by passing a Theme table:

local window = KyriLib.new("My Script", {
    GameName = "MyGame",
    Theme = {
        bg = Color3.fromRGB(10, 10, 15),
        accent = Color3.fromRGB(255, 100, 100),
        text = Color3.fromRGB(255, 255, 255)
    }
})
Info

You only need to specify the colors you want to change. Any colors not specified will use the default theme.

Change Accent Color

window:accent(Color3.fromRGB(255, 100, 100))

Tabs

Creating a Tab

window:tab(name, icon)

Parameters

Parameter Type Description
name string Tab name
icon string (optional) Lucide icon name

Example with Icons

local weapons = window:tab("Weapons", "sword")
local settings = window:tab("Settings", "settings")
Icon Names

Available icons: sword, shield, user, settings, move, music, home, star, heart, and more. See lucide.dev for all icon names.

Elements

Button

tab:button(text, callback)

Creates a clickable button.

Toggle

tab:toggle(text, default, callback, flag)

Creates a toggle switch. If a flag is provided, the state is saved to configs.

Slider

tab:slider(text, min, max, default, callback, flag)

Creates a slider for number values.

Input

tab:input(text, placeholder, callback, flag)

Creates a text input box.

Dropdown

tab:dropdown(text, options, default, callback)

Creates a dropdown menu with selectable options.

Warning

Dropdowns do not save to configs automatically to avoid bugs.

Label

tab:label(text)

Creates a text label for organizing sections.

Example

local main = window:tab("Main")

main:label("player options")

main:button("Reset", function()
    print("Reset clicked")
end)

main:toggle("Fly", false, function(state)
    print("Fly:", state)
end, "fly")

main:slider("Speed", 16, 200, 16, function(value)
    print("Speed:", value)
end, "speed")

main:input("Username", "Enter name", function(text)
    print("Name:", text)
end, "username")

main:dropdown("Location", {"Spawn", "Base", "Shop"}, "Spawn", function(value)
    print("Selected:", value)
end)

Notifications

Send Notification

window:notify(title, text, duration)

Parameters

Parameter Type Description
title string Notification title
text string Notification message
duration number Duration in seconds (default: 3)

Example

window:notify("Success", "Feature enabled", 2)
window:notify("Error", "Something went wrong", 5)

Config System

KyriLib automatically saves and loads settings for elements with flags. A Settings tab is automatically created with save/load functionality.

Accessing Flags

-- Get value
local speed = window.flags.speed_flag

-- Set value programmatically
window.flags.fly_flag_set(true, true)
Info

Configs are saved per-game using the GameName option. Users can save multiple configs with different names.

Complete Example

local KyriLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/Justanewplayer19/KyriLib/refs/heads/main/source.lua"))()

local window = KyriLib.new("prison life", {
    GameName = "PrisonLife"
})

local main = window:tab("Main", "home")
local combat = window:tab("Combat", "sword")

main:label("movement")

main:toggle("fly", false, function(state)
    if state then
        window:notify("Movement", "Fly enabled", 2)
        -- fly code here
    else
        window:notify("Movement", "Fly disabled", 2)
        -- stop fly
    end
end, "fly_toggle")

main:slider("walkspeed", 16, 200, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end, "walkspeed")

main:dropdown("teleport to", {"spawn", "prison", "criminal base"}, "spawn", function(value)
    window:notify("Teleport", "Teleported to " .. value, 2)
    -- teleport code here
end)

combat:label("weapons")

combat:button("give all weapons", function()
    window:notify("Weapons", "Gave all weapons", 2)
    -- weapon code here
end)

combat:dropdown("gun mode", {"normal", "rapid fire", "one shot"}, "normal", function(value)
    window:notify("Gun Mode", "Set to " .. value, 2)
    -- gun mod code here
end)

window:accent(Color3.fromRGB(138, 116, 249))
